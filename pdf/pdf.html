splitPDF

Divida seus arquivos PDF em arquivos menores automaticamente e determine o tamanho máximo das partes!

Link para download

Como usar:

Realizar o donwload pelo link acima e extrair o arquivo baixado com WinRAR ou semelhantes.

Após a descompactação, você terá uma pasta chamada splitPDF. Dentro dela, haverá um arquivo chamado splitPDF e uma pasta chamada docs.

Coloque os arquivos PDF que deseja dividir dentro da pasta docs e execute o arquivo splitPDF.

Uma tela preta aparecerá em sua tela. Aguarde até que a mensagem "Tamanho máximo de cada arquivo (em MB):" apareça.

Digite o tamanho máximo que cada arquivo de saída poderá ter, em MB, e aperte enter. Somente são aceitos números inteiros positivos.

Aguarde a execução do programa. Dependendo do tamanho e da quantidade de arquivos ou da velocidade de seu computador, isso pode levar de alguns segundos até poucos minutos.

Se a tela preta fechar sozinha, é provável que algum erro tenha acontecido. Veja na seção abaixo os erros mais comuns e como evitá-los.

Se o programa terminar sua execução normalmente, a mensagem "Aperte enter para sair" aparecerá. Aperte enter e a tela preta fechará.

Vá até a pasta docs. Nela, além dos documentos originais, deve haver uma nova pasta para cada arquivo. Dentro de cada uma dessas pastas estarão as partes dos arquivos divididos.

As pastas criadas terão o mesmo nome dos arquivos originais, e cada parte do arquivo dividido terá o sufixo -parteN, onde N é o número da parte.

Dica:

Você pode mover o splitPDF para qualquer pasta de seu computador e executá-lo a partir dessa pasta. O programa, sempre que executado, procurará por uma pasta chamada docs dentro do diretório em que estiver no momento e dividirá todos os arquivos com extensão .pdf que estiverem diretamente dentro daquele diretório.

Erros comuns:

  • Durante a execução do programa ele criará pastas, dentro da pasta 'docs', com o mesmo nome dos arquivos .pdf que estiverem dentro da pasta 'docs'. Se essas pastas já existirem, não será possível para o programa cria-las e ele falhará a execução, sem dividir nenhum dos arquivos. Esta escolha foi feita para evitar que arquivos sejam sobrescritos por acidente. Para evitar erros, garanta que não existe nenhuma pasta com o mesmo nome dos arquivos .pdf que estiverem dentro da pasta 'docs'.
  • Arquivos .pdf gerados a partir de fotos tiradas por smartphones podem ter tamanhos muito grandes por página, o que pode atrapalhar a execução do programa. O mesmo pode acontecer com imagens scaneadas em resolução muito grande, ou arquivos .pdf com muitas imagens. Caso os arquivos de saída apresentem inconsistências em seus tamanhos ou sejam gerados vários arquivos com uma única página cada, tente comprimir os arquivos de entrada para reduzir a resolução e o tamanho de cada página.
  • Há um bug conhecido com arquivos relativamente muito pequenos (em que cada página é muito pequena), que pode fazer o programa rodar para sempre e não executar as divisões. Caso o programe rode para sempre, feche a tela preta do terminal que abriu para parar sua execução, ou, na tela do terminal, aperte ctrl+c.
